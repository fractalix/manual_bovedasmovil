Conceptos[inicio]

  • Computadora móvil
    • Por computadora móvil o móvil en adelante, nos referimos a la computadora que ejecuta la aplicación móvil para la ejecución de las tareas.
  • Aplicación móvil
    • La aplicación móvil es la aplicación desarrollada para apoyar al área de bóvedas en las labores relacionadas con el manejo de los materiales fílmicos. Es la aplicación móvil la que realiza la sincronización, donde se ejecutan las tareas y desde donde se entregan los resultados.
  • Sincronización
    • La sincronización la entendemos como el proceso de intercambio de información entre la aplicación móvil y el servidor central. Es durante este intercambio que las tareas se decargan al móvil y los resultados se envían al servidor central. También se realiza intercambio de información adicional, como datos de autenticación de usuarios, información de nichos y bóvedas, entre otros.
  • Tarea
    • Por tarea entendemos un trabajo o actividad que el almacenista tiene que realizar en las bóvedas utilizando la móvil. Las tareas se crean en el sistema central y son descargadas al móvil para su ejecución por el almacenista al que se asignó la tarea.
  • Resultado
    • Un resultado lo entendemos como la información obtenida de la realización de la tarea por el almacenista. Esta información es recabada con la móvil y es enviada al sistema central una vez que el almacenista la ha marcado como concluída.

Computadora móvil[inicio]

Como parte del CLAF, se provee del equipo y el software para que el ingreso, recolección, y en general los movimientos de material desde, en y hacia las bóvedas sea más ágil y más confiable. El presente manual describe las operaciones y la interfaz de la aplicación desarrollada para el dispositivo móvil Intermec CK3 basado en Windows Mobile 6.

El dispositivo móvil cuenta con un escáner de códigos de barra bidimensionales; éste escáner es activado mediante el gatillo localizado en el mango. Se tiene también la opción de operar el escáner vía el botón que se encuentra entre el teclado y la pantalla.

Para poder hacer uso de las funciones en la aplicación se requiere que el dispositivo se comunique a un servidor para realizar una "sincronización" de datos. Al momento la forma de comunicar la aplicación en el dispositivo móvil con el servidor es a través de una base que conecta el dispositivo con una computadora personal o laptop vía USB. Esta computadora debe tener acceso a la red. Además de esto se requiere del software ActiveSync o equivalente, por lo que la computadora deberá ejecutar el sistema operativo Windows XP o superior.

Aplicación móvil[inicio]

El nombre de la aplicación es "BovedasMovil". Luego de la instalación hay un acceso en el menú de inicio.


Si la computadora móvil puede establecer comunicación con el servidor central al inicio, ésta actualizará la lista de usuarios y las tareas de verificación nuevas. Cuando sucede esto se informa al usuario mediante un mensaje.


Si el usuario ingresa datos de autenticación incorrectos se muestra el mensaje indicándolo.


Una vez que se ingresan los datos de autenticación correstamente, se muestra el menú inicial con tres opciones: Tareas, Sincronizar y Utilerías. El botón de Sincronización estará activo o inactivo dependiendo de si existe conexión con una PC con el ActiveSync previamente configurado.
